About me

This blog documents my work in offensive security, with a focus on web exploitation and Active Directory red teaming. Each post aims to break down a vulnerability clearly enough that the reasoning behind the exploit is as useful as the exploit itself. It then notes the toolset (Nmap, Burp Suite, Metasploit, Hashcat, CrackMapExec, Wireshark, BloodHound) across Linux and Windows targets, and closes on the goal of operating as a red team penetration tester.
